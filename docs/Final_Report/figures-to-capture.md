## Capture Notes

- Use demo-safe numbers only.
- If location is visible, use non-sensitive demo traces or blur the map/link as needed.
- Prefer consistent device orientation and theme across screenshots.
- Capture screenshots after the app is configured, not in half-complete states, unless the figure is explicitly about missing-permission behavior.
- For diagrams, export high-resolution images so they remain readable in a printed PDF.
