App Source (Flutter)

This is a minimal Flutter MVP scaffold:
- Start/Stop trip
- Simulated crash alert flow
- Bottom navigation: Home / Lab / Results
- Splash screen + logo branding
- Scenario lab with repeatable sensor playback
- Live accelerometer capture from Android device sensors
- Saved trace replay with automatic PASS/CHECK evaluation
- Batch scenario runner for quick evaluation
- Adjustable detection thresholds
- Persistent event log with JSON export
- CSV export for analysis
- Results summary screen with report generation
- Report pack export (Markdown + CSV)
- JSON export and charted results
- Printable report template + report pack generator script
- HTML report export (print-to-PDF friendly)
- Final report pack (MD + HTML + CSV + JSON + diagrams + chart)

To run (after Flutter SDK is installed):
- flutter pub get
- flutter run
